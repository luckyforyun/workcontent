# 一键SOS 紧急求助网页应用 (One-click SOS Web App)

## 📌 项目介绍
「一键SOS」是一个专为移动端设计的、具备沉浸式体验和极简交互的单页前端应用（SPA）。本项目重点在于模拟在紧急情况下，帮助用户快速、准确、误触率极低地发送求救信号。

项目采用了极简的**黑红高对比度暗黑模式 (Dark Mode)** 设计，从交互动画到色彩心理学都针对紧急求生场景进行了深度定制。

### ✨ 核心功能亮点
- **🛡️ 1.5秒防误触交互**：首页中心为巨大的红橙色渐变 SOS 按钮。自带呼吸光晕，长按时边缘会出现白色 SVG 环形进度条，蓄满 1.5 秒后方可触发警报，有效防止日常误碰。
- **⏱️ 沉浸式倒计时拦截**：触发后，页面瞬间切换至带有警报感的暗红背景，进入 5 秒倒计时准备发送阶段。用户可以在这段时间内点击「取消求救」撤回操作。
- **🚨 强警示成功状态**：倒计时结束后，界面切换至“救援已在路上”状态，并伴随全局的**红黑循环渐变闪烁**特效，模拟真实的紧急警报灯光，同时提供「拨打 110/119/120」与「快捷短信」的一键直达按钮。
- **📍 实时定位模拟**：展示当前位置经纬度，并在求救触发后附加定位信息（目前为基于浏览器的 Geolocation API 结合模拟数据）。
- **🧭 完整的多页 SPA 架构**：内置丝滑的底栏路由导航，包含：
  - **求救（主页）**：核心触发区与 AED / 视频教程入口。
  - **联系人**：管理紧急情况下的第一通知人。
  - **记录**：查看历史触发的时间与位置记录。
  - **设置**：配置自动化选项（自动短信、电话、循环发送）与求救风险等级。

## 🛠️ 技术栈
- **原生技术**：HTML5, Vanilla JavaScript, 原生 CSS3 变量与 Keyframes 动效。
- **构建工具**：Vite (极速热重载，轻量化)。
- **UI 图标库**：Phosphor Icons (基于 Web Font 的全套矢量图标)。

## 🚀 第一次启动详细步骤

本项目基于 `Node.js` 与 `Vite` 构建，如果您是第一次运行本项目，请按照以下步骤操作：

### 1. 前置环境要求
请确保您的电脑上已经安装了 Node.js 运行环境。
- 您可以在终端执行 `node -v` 检查是否安装。如果没有，请前往 [Node.js 官网](https://nodejs.org/) 下载并安装。

### 2. 进入项目目录
打开您的终端（Terminal / CMD / PowerShell），并切换到本项目的根目录。

### 3. 安装项目依赖
在终端中执行以下命令，让 npm 自动下载 Vite 运行所需的依赖包：
```bash
npm install
```

### 4. 启动本地开发服务器
依赖安装完成后，执行以下命令即可启动项目：
```bash
npm run dev
```

### 5. 在浏览器中预览
终端启动成功后，会输出一个本地预览地址（通常是 `http://localhost:5173/`）。
在浏览器中打开该地址，即可看到完整的「一键SOS」页面。

> **💡 小贴士（最佳体验方式）**：
> 因为本项目专为移动端设计，建议您在浏览器中按下 `F12` 打开开发者工具，点击左上角的 **移动端设备模拟 (Device Toggle)**，选择如 iPhone 14 Pro 等手机型号进行预览，以获得最完美的 UI 和长按交互体验！
